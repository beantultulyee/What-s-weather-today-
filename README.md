# 📅 통합 일정·날씨 캘린더 서비스

## 1. 📌 프로젝트 개요
일정이 있는 날 날씨를 확인하기 위해 별도로 날씨 앱이나 웹사이트를 검색해야 하는 번거로움이 발생합니다. 이 프로젝트는 이러한 불편을 해소하고자 **일정 정보와 해당 날짜의 날씨 정보를 한 번에 확인할 수 있는 통합형 캘린더 서비스**를 제공하고자 기획되었습니다.
사용자는 일정을 보면서 동시에 그 날짜와 날씨를 확인할 수 있으며, 날씨 기반 의상 추천까지 받을 수 있습니다.

## 2. 📂 데이터 설명

### 날씨 Api
- 날짜별 날씨 정보를 제공하는 공공API 사용
- 예보 데이터 기반으로 날짜별 아이콘 및 의상 추천 기능 제공

### 캘린더 Api
- 일정 생성, 조회, 수정, 삭제 등의 기능 제공
- 사용자별 일정 관리 기능

### 데이터 베이스
- 사용자 회원가입 빛 로그인 정보 저장
- 일정 데이터 저장

## 3. 🎨 화면 설계

### 회원가입 페이지
- 신규 사용자 계정 생성
- 필수 정보 입력(아이디, 비번) 후 회원가입 처리

### 메인 페이지
- 달력 UI(캘린더) 제공
- 날짜 선택 시 일정 확인
- 날짜별 날씨 정보 표시
- 일정 등록/수정/삭제 기능
- 의성 추천 표시

## 4. 🛠 기능 목록
- 사용자 **회원가입 및 로그인**
- 캘린더 일정 **등록 / 수정 / 삭제**
- 날짜별 **날씨 아이콘** 표시
- **날씨 기반 맞춤형 의상 추천**
